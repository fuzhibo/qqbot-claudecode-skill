# /qqbot-list

列出所有已配置的 QQ 机器人。

## 用法

```
/qqbot-list
```

## 执行

```bash
qqbot-mcp-cli list
```

## 输出示例

```
📋 已配置的机器人列表

1. my-bot
   状态: ✅ 启用
   AppID: 12345678...
   默认目标: G_123456789
   更新时间: 2025/3/12 18:30:00

2. dev-bot
   状态: ✅ 启用
   AppID: 87654321...
   默认目标: 未设置
   更新时间: 2025/3/11 10:15:00
```

## 说明

显示信息包括：
- 机器人名称
- 启用状态
- AppID（部分显示）
- 默认目标 ID
- 最后更新时间
